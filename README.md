# YouTube Focus Pro

Open-source Chrome extension that removes distractions from YouTube.

## Features

- Hide Shorts
- Hide Comments
- Hide Sidebar
- Subscriptions Only Mode
- Focus Mode

## Tech Stack

- Manifest V3
- Chrome Storage API
- MutationObserver
- DOM manipulation

## Installation

1. Clone repo
2. Open chrome://extensions
3. Enable Developer Mode
4. Click "Load Unpacked"
5. Select folder
